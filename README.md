# Jumbler

Jumbler is a spotify-based app allowing users to generate playlists automatically and helping them to have a deeper listening turnover among artists and songs they like.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
